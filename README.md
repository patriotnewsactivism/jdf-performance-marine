# J.D.F. Performance Marine Web App

A React + Vite + TypeScript site with Tailwind CSS and shadcn-ui.

## Getting started

Prerequisites:
- Node.js 18+ and npm

Install dependencies:
```sh
npm install
```

Run dev server:
```sh
npm run dev
```

Build for production:
```sh
npm run build
```

Preview production build:
```sh
npm run preview
```

## Tech stack
- Vite
- React + TypeScript
- Tailwind CSS
- shadcn-ui
- Supabase (optional integrations)
